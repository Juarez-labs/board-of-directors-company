## What This Is

A public-facing narrative of how we built and operate an AI-agent company. Internal runbooks, ADRs, and agent guides remain in their source locations (`docs/runbooks/`, `docs/decisions/`). This playbook synthesizes those into readable, teachable patterns.

**Audience:** Anyone building their own AI-agent company.

**Credibility rule:** No section is written until the pattern it describes has been proven in this company.
