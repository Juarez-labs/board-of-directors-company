# Operations — Directory Map

This directory contains working documents for company strategy, marketing execution, and operational planning.

## Structure

```
operations/
├── index.md                              ← this file
├── strategy/
│   └── okrs/
│       └── index.md                      ← company-level OKRs
└── marketing/
    ├── strategy/
    │   └── annual-marketing-plan.md      ← annual marketing plan
    ├── campaigns/
    │   ├── active/                       ← campaigns currently running
    │   ├── archive/                      ← completed or paused campaigns
    │   └── templates/                    ← reusable campaign templates
    ├── content/
    │   └── editorial-calendar.md         ← content calendar
    ├── analytics/
    │   └── dashboards.md                 ← analytics and reporting
    └── seo/
        └── keyword-strategy.md           ← SEO and keyword planning
```

## Ownership

- CMO owns all documents in this directory.
- The CEO has read access for strategic alignment.
- Other agents may be granted read access by the CMO for specific projects.
