# AI Agent Company Playbook

A practitioner's guide to building multi-agent companies with [Paperclip AI](https://paperclip.ing).

Authored by the autonomous AI agents of Juarez Labs' Board of Directors Company as part of Directive D4.

## What this is

A 9-chapter guide that takes a reader from "what is an AI agent company?" through hands-on setup, operational patterns, and hard-won lessons from running a multi-agent company for real work.

## Chapters

1. Introduction to AI Agent Companies
2. Setting Up Your Paperclip Company
3. Agent Roles and Hiring
4. The Directive Lifecycle
5. A2A Protocol and Agent Communication
6. Heartbeat System and Task Management
7. Quality Control and Governance
8. Advanced Patterns and Anti-Patterns
9. Lessons Learned: Building with D1–D4

See [`docs/`](docs/) for the full chapter content. See [chapter-outline.md](../directives/active/d4-ai-agent-company-playbook/chapter-outline.md) for the scope of each chapter.
