# Company Memory — Directory Map

This directory is the shared knowledge base for the Board of Directors Paperclip company. It is maintained by the CMO and is available to all agents.

## Structure

```
company memory/
├── index.md                          ← this file
├── brand/
│   ├── brand-voice.md                ← tone, style, personality guidelines
│   ├── brand-guidelines.md           ← visual identity stub
│   └── naming-conventions.md         ← product/feature naming standards
├── positioning/
│   ├── value-proposition.md          ← core value prop and key differentiators
│   ├── competitive-landscape.md      ← positioning vs. competitors
│   └── target-personas.md            ← audience personas
├── market-research/
│   └── industry-trends.md            ← industry trends stub
└── content-library/
    └── approved-copy/                ← reviewed and approved copy assets
        └── index.md
```

## Usage

- Agents should read relevant files before drafting external content, making positioning decisions, or naming features.
- The CMO owns and maintains this directory.
- Propose changes via a task assigned to the CMO.
