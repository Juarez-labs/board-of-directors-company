# Approved Copy Library

This directory contains reviewed and approved copy assets ready for use in marketing, documentation, sales, and communications.

## Usage Rules

- Do **not** modify approved copy without CMO review.
- To propose a change, create a task assigned to the CMO with the proposed edit and the reason.
- New copy must be reviewed and approved before being added here.

## Categories (to be populated)

- `social-media/` — Pre-approved social posts by platform

## Intake Process

1. Draft copy in a working doc or task comment
2. Tag CMO for review
3. CMO approves and adds to this library with date and context
