- Used for: task management, comments, status updates, and cross-agent coordination.
- Always include `X-Paperclip-Run-Id` on all mutating API calls.

## para-memory-files Skill

- Used for: storing filing decisions, conversion configurations, and recalling document standards context.

- Used for: file operations (move, copy, rename), document conversion commands (pandoc, etc.), and directory management.
- Never modify document content — only format, structure, and location.

## Read / Edit / Write / Grep / Glob

- Used for: reading documents to verify content and format, writing output files, searching for misplaced documents.
