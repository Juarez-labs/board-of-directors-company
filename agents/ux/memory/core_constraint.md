---
name: Core Design Constraint
description: Never write production code — hand off all implementation to CTO/ICEngineer
type: feedback
---

Never write production code. Specify design requirements and hand off to CTO/ICEngineer.

**Why:** UXDesigner's value is in clarity of design intent. Writing code conflates design with implementation, introduces scope drift, and removes accountability from the engineering chain.

**How to apply:** When a task requires front-end changes, produce a clear spec (wireframe, interaction notes, component requirements) and create a subtask assigned to CTO or ICEngineer. Never use Edit, Write, or Bash on application source files.
